
#### ðŸ“Œ Explanation:
- `val1 = 200`, `val2 = 400`

- So, `num1 = [500]`

> âœ… **Rest** collects the remaining items into an array (used in function parameters).  
> âœ… **Spread** expands elements (used in function calls, arrays, or objects).

## ðŸ“¦ Handling Objects as Function Parameters

```js

    username: "Hitesh",
    price: 199,
};

function handleObject(anyobject) {
    console.log(\`Username is \${anyobject.username} and price is \${anyobject.price}\`);
}

handleObject(user);
handleObject({ username: "sam", price: 399 });
```

#### ðŸ“Œ Explanation:
- `handleObject` takes an object as an argument.
- You can pass:
  - A variable holding the object (`user`)
  - Or a direct object literal (`{ username: "sam", price: 399 }`)
- This pattern is useful for reusable and flexible function inputs.

## ðŸ“Š Working with Arrays in Functions

```js

function returnSecondValue(getArray) {
    return getArray[1];
}

console.log(returnSecondValue(myNewArray));
console.log(returnSecondValue([200, 400, 100, 600]));
```

#### ðŸ“Œ Explanation:
- Function `returnSecondValue` returns the element at **index 1**.
- Arrays in JavaScript are **zero-indexed**.
- `[200, 400, 100, 600] â†’ 0: 200, 1: 400` â†’ Output: `400`
